(static) angleSet(mac, leftUpper, leftLows, rightUpper, rightLow)

Set BP7S device angle.
  • This is an asynchronous call, it will return immediately.
  • If get successfully, following event will be emit:
    BP7SModule.Event_Notify("event_notify_bp7s")
    The key and value will be as below:
    KeyValue
    actionBPProfileModule.ACTION_SET_ANGLE_SUCCESS_BP("set_angle_success")
  • If error happens, following event will be emit:
    BP7SModule.Event_Notify("event_notify_bp7s")
    The key and value will be as below:
    KeyValue
    actionBPProfileModule.ACTION_ERROR_BP("error_bp")
    BPProfileModule.ERROR_NUM_BP("error")the error of Bp device.
    e.g. 400
    BPProfileModule.ERROR_DESCRIPTION_BP("description")the description of error.
    e.g. the argument of method is illegal
  • Attention, if you want to be notified, it is mandatory to call following method before you call this method:
    DeviceEventEmitter.addListener(BP7SModule.Event_Notify, function (e: Event){});
Parameters:
Name Type Description
mac string Device's mac address
leftUpper int the maximum measure angle of left hand, the maximum value must less than 90.
leftLows int the minimum measure angle of left hand, the minimum value must more than 0, and less than leftUpper.
rightUpper int the maximum measure angle of right hand, the maximum value must less than 90.
rightLow int the minimum measure angle of right hand, the minimum value must more than 0, and less than rightUpper.
