(static) ACTION_ERROR_HS

The action type of callback indicating the error of HS device.
The key and value will be as below:
KeyValue
actionHSProfileModule.ACTION_ERROR_HS("error_hs")
HSProfileModule.ERROR_NUM_HS("error") HSProfileModule.ERROR_ID_ILLEGAL_ARGUMENT indicates parameter error("400")
HSProfileModule.ERROR_ID_WIFI_DISABLED indicates wifi is disabled("500")
HSProfileModule.ERROR_DESCRIPTION_HS("description")The error description data.
Example: getOfflineData() parameter userPstCode should be in the range [0, 19].
Example message:
{
    "error": 400,
    "description": "getOfflineData() parameter userPstCode should be in the range [0, 19]."

KeyValue

2The Scale failed to initialize.
3Maximum weight has been exceeded.
4The Scale can't capture a steady reading.

6Movement while measuring.
7Invalidate.
8Scale memory access error.
9No memory.
10Device disconnect.
11Communication error.
