(static) getOfflineData(mac)

Get the value of historical data in the HS4S.
  • This is an asynchronous call, it will return immediately.
  • If get successfully, following event will be emit:
    HS4SModule.Event_Notify("event_notify_hs4s")
    The key and value will be as below:
    KeyValue
    actionHSProfileModule.ACTION_HISTORICAL_DATA_HS("historicaldata_hs")
    HSProfileModule.HISTORDATA__HS("historyData") HSProfileModule.DATAID("dataID") The data id. Example: 67FFC7A454148723718144A604D4152B
    HSProfileModule.MEASUREMENT_DATE_HS("date") The measure date. Value format: yyyy-MM-dd HH:mm:ss
    HSProfileModule.WEIGHT_HS("value") The key of the weight result. The value's unit is kg. Value range: 0.0-150.0
    HSProfileModule.FAT_HS("fat") The body fat rate. Value range: 0.0-100.0
    HSProfileModule.WATER_HS("water") The body water rate. Value range: 0.0-100.0
    HSProfileModule.MUSCLE_HS("muscle") The body muscle rate. Value range: 0.0-100.0
    HSProfileModule.SKELETON_HS("skeleton") The bone mass. Value range: 0.0-25.5(0xFF/10.0)
    HSProfileModule.FATELEVEL_HS("fatelevel") The fat level. Value range: 0-255(0xFF)
    HSProfileModule.DCI_HS("DCI") he DCI data. Value range: 0-65535(0xFFFF)
  • If error happens, following event will be emit:
    HSProfileModule.Event_Notify("event_notify_hs4s")
    The key and value will be as below:
    KeyValue
    actionHSProfileModule.ACTION_ERROR_HS("error_hs")
    HSProfileModule.ERROR_NUM_HS("error") HSProfileModule.ERROR_ID_ILLEGAL_ARGUMENT indicates parameter error("400")
    HSProfileModule.ERROR_ID_WIFI_DISABLED indicates wifi is disabled("500")
    HSProfileModule.ERROR_DESCRIPTION_HS("description")The error description data.
    Example: getOfflineData() parameter userPstCode should be in the range [0, 19].
  • Attention, if you want to be notified, it is mandatory to call following method before you call this method:
    DeviceEventEmitter.addListener(HSProfileModule.Event_Notify, function (e: Event){});
Parameters:
Name Type Description
mac string Device's mac address

    KeyValue
    actionHSProfileModule.ACTION_LIVEDATA_HS("liveData_hs")
