(static) bindDeviceHS6(birthday, weight, height, isSporter, gender, serialNumber)

Bind the user and scale together,after bind success user's weight datas can be transmitted to the cloud. And this method is a time consuming operation that cannot be called in the main thread.
  • This is an asynchronous call, it will return immediately.
  • If get successfully, following event will be emit:
    HS6Module.Event_Notify("event_notify_hs6")
    The key and value will be as below:
    KeyValue
    actionHS6ProfileModule.ACTION_HS6_BIND("hs6_bind")
    HS6ProfileModule.HS6_BIND_EXTRA("bindData") HS6ProfileModule.BIND_HS6_RESULT("bindResult") The result of bind HS6 device. 1 bind success; 2 the scale has no empty position ;3 bind fail.
    HS6ProfileModule.HS6_MODEL("model") The model of HS6.
    HS6ProfileModule.HS6_POSITION("position") The user position in the HS6 range from 1 to 10, -1 if bind false.
    HS6ProfileModule.HS6_SETTED_WIFI("settedWifi") Whether has setted wifi.1 setted ;0 not
  • If error happens, following event will be emit:
    HS6ProfileModule.Event_Notify("event_notify_hs6")
    The key and value will be as below:
    KeyValue
    actionHS6ProfileModule.ACTION_HS6_ERROR("hs6_error")
    HS6ProfileModule.HS6_ERROR("hs6_error")The error code: 1 user information upload error; 2 bind error.
  • Attention, if you want to be notified, it is mandatory to call following method before you call this method:
    DeviceEventEmitter.addListener(HS6ProfileModule.Event_Notify, function (e: Event){});
Parameters:
Name Type Description
birthday string format like yyyy-MM-dd HH:mm:ss
weight float the unit is kg
height int the unit is cm
isSporter int is sporter; 2 is not ;3 unknown
gender int 0 is male ;1 is female
serialNumber string the mac address of the scale

Get AccessToken of HS6 user,and this method is a time consuming operation that cannot be called in the main thread. After get AccessToken, you can call openApi(http://developer.ihealthlabs.com) to pull data form iHealth cloud.
  • This is an asynchronous call, it will return immediately.
